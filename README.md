# video-player-app

[![Build Status](https://travis-ci.org/m0t0r/video-player-app.svg?branch=master)](https://travis-ci.org/m0t0r/video-player-app)

Installation
------------

```
npm install
npm start
open in a browser: http://localhost:8080
```

Unit tests
------------

Single run:

```
npm test
```

Dev mode:

```
npm run test-dev
```
